Introducción

JavaScript es un lenguaje de programación que permite implementar funcionalidades complejas en páginas web.Es un lenguaje interpretado, orientado a objetos y basado en prototipos. JavaScript es un lenguaje de programación que permite implementar funcionalidades complejas en páginas web.

Cómo funciona

JavaScript se ejecuta en el navegador del cliente, lo que significa que no es necesario enviar solicitudes al servidor para cada acción del usuario. Esto permite una experiencia de usuario más fluida y rápida. Para poder usarlo dentro del navegador, es necesario colocarlo dentro de un etiqueta.

<script>

Esto se puede hacer directamente entrelazando el HTML de dos formas:

<script src="script.js"></script>

o:

 <html>

...head, meta, title...

  <body>

   <script>

    let mensaje = "Hola lectores!!"

    alert(mensaje) //Esto imprime el mensaje que había puesto arriba.

   </script>

  </body>

 </html>

Declaración de variables

  • En JavaScript, existen dos tipos de variables: las variables primitivas y los objetos. Las variables primitivas son aquellas que contienen un valor único y no se pueden modificar. Estos tipos de variables se declaran utilizando la palabra clave const. Por otro lado, los objetos son variables que pueden contener varios valores y pueden ser modificados. Estos tipos de variables se declaran utilizando la palabra clave let. Además, JavaScript tiene una tercera forma de declarar variables que es utilizando la palabra clave var. Sin embargo, esta forma de declarar variables es considerada como anticuada y no se recomienda utilizarla.

    La sintaxis para declarar una variable es la siguiente:

    let nombre = valor;

    Condicionales

    Los condicionales son estructuras de control que nos permiten ejecutar diferentes bloques de código según una condición. En JavaScript, existe el condicional if, que pregunta si una condicion es verdadera, si no esto y si no, lo otro.

    La sintaxis para un if es la siguiente:

    if (condición) {
    // Código a ejecutar
    };

    Bucles

    Los bucles son estructuras de control que nos permiten ejecutar un bloque de código varias veces. En JavaScript, existen tres tipos de bucles: for, while y do while.

    El for funciona yendo de un punto A a un punto B. Por ejemplo, queremos hacer un recorrido en el rango 1 a 10, entonces podemos hacer que vaya haciendo 1, 2, 3, 4, 5, 6, 7, 8, 9 y 10.

    Sintaxis del for:

    for (inicialización; condición; incremento) {
    // Código a ejecutar
    };

    El while funciona MIENTRAS (de ahí el while) la condición se cumpla, es decir, sea true

    Sintaxis del while:

    while (condición) {
    // Código a ejecutar
    };

    El do while funciona por lo menos una vez, al finalizar el bloque pregunta si se sigue cumpliendo dicha función para volver a repetirla.

    La sintaxis para un do while es la siguiente:

    do {
    // Código a ejecutar
    } while (condición);

    Funciones

    Las funciones son bloques de código que se pueden llamar varias veces con diferentes parámetros.

    La sintaxis para declarar una función es la siguiente:

    function nombre (parámetros) {
    // Código a ejecutar
    };

    La sintaxis para llamar a una función es la siguiente:

    nombre (parámetros);

    Eventos

    Los eventos son acciones que ocurren en el navegador y que pueden ser detectadas por JavaScript.

    Existen varios tipos de eventos, como el evento click, el evento submit, el evento load, entre otros.

    La sintaxis para detectar un evento es la siguiente:

    elemento.addEventListener("evento", función);

    React

    React es una biblioteca de JavaScript para construir interfaces de usuario. React se basa en componentes, que son bloques de código que se pueden reutilizar.

    La sintaxis para declarar un componente es la siguiente:

    const Nombre = (props) => {
     return (
      <div>Contenido</div>
     );
    };

    La sintaxis para llamar a un componente es la siguiente:

    <Nombre />

    Nodejs

    Nodejs es un entorno de ejecución de JavaScript que permite ejecutarlo en el lado del servidor. Nodejs utiliza el motor de JavaScript V8 de Google Chrome y es de código abierto. Nodejs proporciona una API para interactuar con el sistema operativo y los recursos del sistema.

    La sintaxis para importar un módulo en Nodejs es la siguiente:

    const módulo = require("módulo");
